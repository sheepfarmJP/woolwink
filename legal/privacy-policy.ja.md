# Woolwink プライバシーポリシー

最終更新日: 2026-06-05

## 1. 基本方針 — Woolwink は個人データを収集しません

Woolwink（以下「本ソフトウェア」）は、あなたの**画面キャプチャ・音声・入力内容を含む一切の個人データを、収集・送信・保存しません。**

- すべての処理（画面キャプチャ、ローカル音声認識）は、**あなたのPCの中だけ**で完結します。
- 私たちは**サーバーを運用しておらず**、テレメトリ・利用解析・トラッキング・広告識別子を一切使用しません。
- ネットワーク接続を遮断しても、本ソフトウェアの基本機能は動作します。

## 2. 端末内での処理

- **音声認識**はあなたのPC上のローカルモデル（Whisper）で実行され、音声が外部へ送信されることはありません。
- **設定ファイル**（config.json）と**監査ログ**（送信の記録）は、あなたのPC内にのみ保存されます。
- 監査ログに記録されるのは「送信の種別・日時・画像枚数・文字数」のみで、**画像や本文の内容は保存しません。**

## 2.5 ネットワーク通信について（完全な透明性）

本ソフトウェアがインターネットに接続するのは、次の **2 つの場合だけ**です。**いずれもあなたの画面・音声・入力内容を送信しません。**

1. **アップデート確認（任意・オフにできます）。** GitHub 上の小さな情報ファイル（バージョン番号のみ）を取得します。リクエストにあなたを識別する情報は含まれません。自動確認は設定でオフにでき、更新 URL を空にすれば完全に無効化できます。
2. **音声認識モデルの初回ダウンロード。** 音声機能を初めて使うとき、完全ローカルの音声認識に使うオープンソースの Whisper モデルを、配布元（Hugging Face）から一度だけダウンロードします。以後はオフラインで動作し、このダウンロード時にあなたの音声が送信されることはありません。

これら 2 つ以外に、本ソフトウェアが外部へ通信することはありません（テレメトリ・解析・トラッキングは一切ありません）。

## 3. Claude Desktop（Anthropic 社）への受け渡し

- 本ソフトウェアは、あなたが選んだ内容を、あなたのPCにインストールされた **Claude Desktop の入力欄** に渡します。
- 受け渡した後の内容の取り扱いは、**Anthropic 社のプライバシーポリシー・利用規約**に従います。本ソフトウェアはその処理の当事者ではありません。
  - Anthropic プライバシーポリシー: https://www.anthropic.com/legal/privacy
  - Anthropic 消費者規約: https://www.anthropic.com/legal/consumer-terms

## 4. 第三者への提供

本ソフトウェアの開発者は、あなたのデータを**受け取らず、第三者にも提供しません**（開発者が保持するデータが存在しないため）。

## 5. 権限（マイク・画面・クリップボード）

本ソフトウェアは、機能の実行にあたりマイク・画面キャプチャ・クリップボードへアクセスします。これらは上記の目的（端末内処理と Claude Desktop への受け渡し）にのみ使用されます。マイクの許可は Windows の設定（プライバシー → マイク）でいつでも変更できます。

## 6. （将来）有料版（Pro）について

Pro 版の決済を行う際は、決済処理サービス **Stripe** が氏名・メールアドレス・決済情報を取り扱います（Stripe のプライバシーポリシーに従います）。ライセンスの発行・照合のため、**復元不可能な端末ID（ハッシュ値）**とメールアドレスを購入時に扱います。端末IDは端末の識別のみに用い、あなたの画面・音声・入力内容は一切含みません。その際は本ポリシーを更新し、**対象データ・利用目的・保持期間・あなたの権利**を明記します。
